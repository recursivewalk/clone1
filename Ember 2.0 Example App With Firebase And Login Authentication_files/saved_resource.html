                                                  Adriaan a year ago

                                                  Awesome post Erik. One minor detail: I see you are using a lot of `let` in ES6 instead of `const`, for example inside the `save` actions. You can use `const` there and I think that is a common standard to use `const` over `let`. In a loop for example you can define a variable with `const` and loop over the definition multiple times. The same goes for functions (like `save`) you call multiple times.

                                                  I learned some nice things about `mut` variables and the `features` folder. Thanks for that!
